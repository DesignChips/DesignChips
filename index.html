仕事でやっていること

仕事では、主にWebフロントエンドの設計・開発を行うことが多いです。新規サービスの開発では、技術選定やデプロイ自動化、開発環境の構築から実装までを行っていました。既存サービスの開発では、フロントエンドの開発基盤の改善や新機能の実装を行っています。

関わっているサービスの分野は衣食住が、サービスの形態としてはTwo-sided Platformが大半です。オンラインのみで完結というよりはオフラインの行動に結びつくようなサービスの開発に関わっていることが多いです。

技術的には以下のような分野に仕事で関わっています。
HTML / CSS/ JavaScript / Testing / Build Tools / Lint Tools / Package Managers / Firebase / CI/CD
